Probar geolocalización
  1. En la raíz del repo: npm run build
  2. Aquí: npm run android o npm run ios
  3. Usa el preset test público o sirve geolocation-test.html por HTTPS (ngrok)
  4. Con debug: true revisa Logcat (Android) o Xcode console (iOS)
